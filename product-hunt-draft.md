## Longer Description

I built ÚČąšŹŞŠúő Classic because I played Dou Shou Qi / Jungle Chess when I was young, and wanted to bring back that childhood feeling.

Many versions online feel more like serious chess apps. I wanted to make a cuter version that children and families can enjoy, while still keeping the classic strategy: animal ranks, tactical captures, rivers, traps, and dens.
