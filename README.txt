This repository publishes the public Animal Chess support, marketing, and privacy pages through GitHub Pages.

Required App Store URLs currently configured in metadata:
- Marketing: https://williamlock.github.io/Animal-Chess-Site/
- Support: https://williamlock.github.io/Animal-Chess-Site/support.html
- Privacy Policy: https://williamlock.github.io/Animal-Chess-Site/privacy.html
- Source/support repository: https://github.com/WilliamLock/Animal-Chess

After hosting, run:
  scripts/configure_public_release_urls.sh https://your-public-host/path/
  scripts/check_public_release_urls.sh
